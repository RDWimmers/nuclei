Getting started

New to nuclei? Don’t worry, you’ve found the perfect place to get started!

Installation

nuclei can be installed by running

pip install cems-nuclei

Please note that to use GeoDataFrames from the geopandas library cems-nuclei[geo] should be installed.

Basic usage

To have a look at the available API’s in NUCLEI run the following code-block:

In [1]: import nuclei

In [2]: print(nuclei.get_applications())
['soil-reinforcement', 'xettlement', 'gef-model', 'cross-section', 'gef-map', 'reports', 'piles', 'vibracore']

For more information about the available endpoints of single API run the following code-block:

In [3]: print(nuclei.get_endpoints("piles"))
['/compression/multiple-cpts/plot-bearing-capacity', '/compression/multiple-cpts/report', '/compression/multiple-cpts/results', '/compression/single-cpt/plot-bearing-capacity', '/compression/single-cpt/report', '/compression/single-cpt/results', '/friction-ranges/lower-bound', '/get-task-result', '/grouper/group_cpts', '/grouper/group_metrics', '/grouper/optimize_groups', '/healthcheck', '/pile_properties/shape', '/pile_properties/type']

To call an endpoint use nuclei.api_zoo.call_endpoint()

To help the user create a schema to use in the API call nuclei has a nuclei.utils.python_types_to_message() function. This function transforms python types to JSON types.

The following code-block creates a pandas.DataFrame and transforms it to a serialized DataFrame.

In [4]: import pandas as pd

In [5]: schema = pd.get_dummies(pd.Series(list('abcaa')))

In [6]: print(nuclei.utils.python_types_to_message(schema))
{'nuclei': {'type': 'PANDAS_DataFrame'}, 'body': '{"a":{"0":1,"1":0,"2":0,"3":1,"4":1},"b":{"0":0,"1":1,"2":0,"3":0,"4":0},"c":{"0":0,"1":0,"2":1,"3":0,"4":0}}'}

To transform a serialized DataFrame to its original type use the nuclei.utils.python_types_to_message() function.

In [7]: message = nuclei.utils.python_types_to_message(schema)

In [8]: print(nuclei.utils.message_to_python_types(message))
   a  b  c
0  1  0  0
1  0  1  0
2  0  0  1
3  1  0  0
4  1  0  0

Please note that a list that of python types will be serialized as well. This means that the individual object will be transformed.

In [9]: message = nuclei.utils.python_types_to_message(schema)

In [10]: print(nuclei.utils.message_to_python_types([{"a": 12}, message]))
[{'a': 12},    a  b  c
0  1  0  0
1  0  1  0
2  0  0  1
3  1  0  0
4  1  0  0]
